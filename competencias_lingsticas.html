COMPETENCIAS LINGÜÍSTICAS

Como competencia se conoce a aquellas habilidades, capacidades y conocimientos que una persona tiene para cumplir eficientemente determinada tarea. En este caso, nos centraremos en el área del lenguaje: 

 Se establecen seis funciones esenciales inherentes a todo proceso de comunicación: emotiva, apelativa, referencial, metalingüística, fática y poética.

Estas funciones se enmarcan dentro de las competencias que posee el hablante para usar la lengua. Entre ellas se destacan la competencia lingüística, paralingüística, comunicativa y cultural.
